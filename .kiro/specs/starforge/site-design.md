# starforge — 公開作品化（Site Deep-Dive）Design

> 現状の単一サンドボックスMVP（`main/gpu/simulation/renderer/camera` + `nbody.wgsl/render.wgsl`）を、
> 「展示室 → 全画面没入」のシネマティックN体作品集へ拡張する。**ローカル動作のみ・デプロイなし。**

## 1. 体験フロー
- **入口＝展示室（Gallery）**：6シーンを黒地・グロー付きの美しいタイルで並べる。
- タイルクリック → **全画面没入（Full-bleed）**でそのシーンが立ち上がる（短いチラ見せ/フェードイン）。
- 没入中の操作：ドラッグで天体追加（ドラッグ方向に初速・矢印オーバーレイ）／ホイールでカーソル方向ズーム／右・Shiftドラッグでパン／Space一時停止／R リセット／**Escで展示室へ戻る**。
- WebGPU非対応時：既存フォールバック文言を踏襲。

## 2. 収録シーン（6）
各シーンは「初期条件プリセット（bodyバッファ生成）＋カメラ初期値＋短い説明文」を持つ。
| id | 名称 | 初期条件の要点 | 見どころ |
|---|---|---|---|
| solar | 実太陽系 | 中心に大質量＋数個の惑星を円軌道速度で配置 | 安定軌道・Keplerリズム |
| galaxy | 銀河衝突 | 2つの回転円盤（中心質量＋周回粒子）を接近軌道で | 潮汐の尾・最大スペクタクル |
| accretion | 降着円盤 | 中心大質量＋わずかに角運動量を持つ粒子雲 | 螺旋落下・輝くリング |
| collapse | 誕生/重力崩壊 | 低速で拡散した粒子雲（初速ほぼ0） | 自己重力収縮→構造形成 |
| slingshot | 重力スリングショット | 安定系＋高速の侵入天体1つ | 加速・散乱のドラマ |
| cluster | 球状星団 | 球状に数千体、ビリアル平衡に近い速度分散 | 多体凝集・GPUの本領 |

## 3. ビジュアル署名（採用）
- **Bloom**（コア発光・土台）
- **軌道トレイル**（残光の航跡）：履歴バッファを毎フレーム減衰合成（前フレームに `alpha<1` を乗じて累積）。
- **星雲バックドロップ**：淡い色ガス（手続き的 or 低解像度ノイズテクスチャ）を最背面に。
- 速度/質量カラーは安価なので任意追加可（今回スコープ外だが拡張余地として残す）。
- **後続ポリッシュ枠（今回スコープ外）**：フィルム仕上げ（ビネット/グレイン/色収差）、ゆっくり寄るシネマカメラ。

## 4. アーキテクチャ（現MVPからの差分）
- 追加モジュール：
  - `src/scenes/` — 各シーンの初期条件生成（`solar.js` 等 or データ駆動の `scenes.js`）。bodyバッファ（位置/速度/質量）とカメラ初期値・説明を返す。
  - `src/gallery.js` — 展示室UI（タイル一覧）と没入ビューの遷移・ルーティング（`#scene=galaxy` 等のハッシュで状態保持）。
  - `src/postfx.js` — bloom（明部抽出→ブラー→加算合成）と trails（履歴バッファ減衰合成）のレンダーパス。追加のオフスクリーンレンダーターゲットを管理。
  - `src/nebula.js`（or 背景パス） — 最背面の星雲描画。
  - 追加WGSL：`bloom.wgsl`（しきい値抽出＋ブラー）、`composite.wgsl`（合成）、`nebula.wgsl`。
- 既存 `renderer.js` は「シーン描画 → postfx 合成」の多段パイプラインに拡張。
- トレイル方式：point描画を履歴RTに累積（クリアせず減衰）。リセット/シーン切替時に履歴クリア。

## 5. スコープ
- **今回（深掘りMVP）**：展示室UI ＋ 6シーンのプリセット ＋ 3効果（bloom/trails/nebula）＋ Escで戻る遷移。
- **今回外**：速度カラー・DoF・フィルム仕上げ・シネマカメラ・シーンのURL共有/保存・実DEM級の重い演算最適化（Barnes-Hut等）。

## 6. 検証（ローカルのみ）
- 全 `.wgsl` を `naga` で検証。
- 全 `.js` を `node --check`。
- 各シーンの初期条件生成をNodeで単体テスト（body数・質量合計・速度分散が想定範囲、NaN無し）。
- postfx/各パスのバインドグループ ↔ WGSL `@group/@binding` 整合をチェック。
- 実描画・トレイル・bloomの見た目は**手動目視**（`python3 -m http.server` でローカル起動）。
- **デプロイ・publish・git commit はしない。**
